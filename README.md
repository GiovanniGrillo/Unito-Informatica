# Repository Università degli Studi di Torino - Informatica

Questo repository raccoglie i progetti e gli esercizi svolti durante il corso di laurea in Informatica presso l'Università degli Studi di Torino, anno accademico 2023/2024. Il repository è organizzato in cartelle separate per ogni corso, ciascuna contenente il proprio materiale didattico, codice sorgente ed esercitazioni.

## Struttura del Repository

### Linguaggi Formali e Traduttori
Corso dedicato allo studio dei linguaggi formali, parser e traduttori. Il progetto include:
- Implementazione di lexer e parser in Java
- Esercizi progressivi dall'analisi lessicale alla generazione di codice
- Utilizzo di strumenti come jasmin.jar per la generazione di bytecode
- [Maggiori dettagli nel README del corso](./Linguaggi_Formali_e_Traduttori/README.md)

### Sistemi Operativi
Progetto di laboratorio incentrato sui sistemi operativi, con focus su:
- Implementazione di componenti in linguaggio C
- Gestione di processi e comunicazione inter-processo
- Configurazioni per diversi scenari (blackout, explode, meltdown, timeout)
- Librerie personalizzate per ogni componente del sistema

### Tecnologie Web
Progetto full-stack con architettura client-server:
- **Client**: Applicazione frontend sviluppata con tecnologie moderne
  - Framework TypeScript/JavaScript
  - Gestione pacchetti con npm
  - Build system Vite

- **Server**: Backend robusto
  - Implementazione Java con Spring Boot
  - Gestione dipendenze con Maven

## Requisiti Generali

- **Java**: JDK installato per Linguaggi Formali e Traduttori
- **C**: Compilatore C per Sistemi Operativi
- **Node.js e npm**: Per il frontend di Tecnologie Web
- **Maven**: Per il backend di Tecnologie Web
